# 📈 Stock Charts 101 — A Beginner's Guide

A beginner-friendly, interactive website that explains how stock market charts work. Built as a college project using only **HTML, CSS, and JavaScript** — no frameworks, no backend.

## 🌐 Live Demo
> _(https://github.com/harshaldhonge2007-sudo/stock-charts-101)_

## 📚 What's Covered

| Section | Topic |
|--------|-------|
| 01 | What is a Stock Chart? |
| 02 | Types of Charts (Line, Bar, Area) |
| 03 | Candlestick Charts (Interactive) |
| 04 | Technical Indicators (MA, Volume, RSI) |
| 05 | Quiz — Test Your Knowledge |
| 06 | Glossary of Key Terms |

## ✨ Features

- **Interactive charts** — switch between Line, Bar, and Area chart types
- **Live candlestick chart** — hover over candles to see OHLC values
- **Indicator toggles** — turn Moving Average, Volume, and RSI on/off
- **Built-in quiz** — 6 questions with instant feedback
- **Glossary** — 12 essential stock market terms explained simply
- **Ticker bar** — scrolling stock prices at the top

## 🛠️ Tech Stack

- HTML5 (Canvas API for candlestick chart)
- CSS3 (animations, responsive layout)
- Vanilla JavaScript
- [Chart.js](https://www.chartjs.org/) via CDN (for line/bar/area charts)

## 🚀 How to Run

Just open `stock-charts-guide.html` in any browser. No installation needed.

```bash
git clone https://github.com/harshaldhonge2007-sudo/stock-charts-101.git
cd stock-charts-101
open stock-charts-guide.html
```
